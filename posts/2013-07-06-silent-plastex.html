On July 6, 2013

Which is obviously undesirable if you’re using it on the commandline. At first I assumed it was just printing to stdout and stderr but trashing those didn’t help. As this wasn’t noted in the documentation, I did a bit of digging and found it was the logging module that was printing out all of this. If you want to disable logging just add the following:

from plasTeX.Logging import disableLogging
disableLogging()

and all of the output from using PlasTeX will be silenced.
